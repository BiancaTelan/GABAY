# GABAY

GABAY: A Web-Based Schedule Reservation System for Outpatient Services in Cainta Municipal Hospital
General-to-specialty Appointment Booking Assistant for You is an alternative option for reserving consultation schedules at Cainta Municipal Hospital

This study aims to address the problem of long waits at Cainta Municipal Hospital 
and the struggles that the staff are facing. The hospital relies on an old-fashioned, manual system where patients must walk in to get an appointment.

The objective of this research is to build a digital assistant that simplifies the patient's journey. It helps guide them smoothly from a 
general check-up to specialized care. This system ensures that appointments are easy to access, based on real data, and verified by the hospital staff.
